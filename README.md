# Генератор команды /summon — Minecraft 1.21.5+

Веб-инструмент для создания команд `/summon` в Minecraft Java Edition (1.21.5 и новее). Полностью на русском языке.

## Возможности

- **Все мобы игры** — 80+ существ, сгруппированных по категориям (враждебные, боссы, нейтральные, мирные, прочее)
- **Снаряжение** — все 6 слотов: шлем, нагрудник, штаны, ботинки, правая рука, левая рука
- **Зачарования** — список меняется под выбранный предмет, уровни от 1 до 255
- **Характеристики** — HP, скорость, броня, урон, сопротивление отбрасыванию, масштаб
- **Особенности** — ребёнок, неуязвимый, бесшумный, без ИИ, горящий/негорящий, светящийся, без гравитации, не исчезает
- **Эффекты зелий** — 32 эффекта с настройкой силы и длительности
- **Имя** — с выбором цвета, жирным и курсивным начертанием
- **Пассажиры** — моб верхом на мобе, любая глубина вложенности
- Команда обновляется автоматически в реальном времени
- Предупреждение, если команда длиннее 256 символов (нужен командный блок)

## Как выложить на GitHub Pages

1. Заходишь на [github.com](https://github.com), регистрируешься если нет аккаунта
2. Кнопка **New repository** (зелёная, справа сверху)
3. Имя репозитория любое, например `mc-summon`. Поставь галочку **Public**. Создать.
4. На странице пустого репозитория → **uploading an existing file** (ссылка посреди экрана)
5. Перетащи файл `index.html` в окно, нажми **Commit changes**
6. Открой вкладку **Settings** (верхняя панель репозитория)
7. Слева выбери **Pages**
8. В разделе **Source** выбери ветку `main`, папка `/ (root)`, нажми **Save**
9. Подожди 1–2 минуты, обнови страницу — наверху появится зелёная плашка с адресом вида `https://твой-логин.github.io/mc-summon/`
10. Эту ссылку можно отправить сыну, открывать с телефона, планшета, компьютера — всё работает

## Как пользоваться

1. Выбрать моба в самом верху
2. Настроить всё, что нужно (имя, характеристики, снаряжение, эффекты, пассажиры)
3. Команда появляется снизу в зелёном окошке
4. Нажать **Копировать**
5. В игре открыть чат (T или /) и вставить
6. Если команда длиннее 256 символов — положить в командный блок:
   - `/give @s command_block` в чате
   - Поставить блок, кликнуть правой кнопкой
   - Вставить команду, нажать **Готово**
   - Активировать рычагом, кнопкой или нажимной пластиной

## Локально (без интернета)

Просто открыть файл `index.html` двойным кликом — запустится в браузере, всё работает без сервера.
